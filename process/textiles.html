Making a Book Loom

Ana Mahadevan
Week One

Vision : I wanted to have my weave kind of match/coincide with the book I was using as my loom, not just aesthetically but also with the themes within the story.

Process Note 1 : I realized pretty quickly that the string for the warp needed to be pretty tight or it wouldn't create enough tension to lift the alternating strings. give it a little more room.

Process Note 2: Set up was initially relatively painless. After a couple times rethreading the string harness, I was to create a clean weave and pull through the semi conductive thread.

Process Note 3: This is where things start to go sideways... I decided I want to try to make the design look like the eye that's on the cover of the book. It seemed like a novel idea initially, but not being able to use the string harness for lines pulled entire weave tighter than I wanted. Seemingly for nothing too, as the design wasn't especially evident.

Here is the finished swatch, after doing some hand-weaving everything was a a bit tighter and not as even. So, even when using the basket weave or just the string harness itself towards the end it was still messy.

Reflection : This was my first time ever weaving anything! I had a lot of fun, but even as I was going through there was so many mistakes I noticed and was very tempted to (and occasionally did) completely unravel.

Using the Cricket Loom

Ana Mahadevan
Week Two

Vision : This week, my friend and I went on a thrifting binge looking for weird and interesting yarns. My plan here was not to make anything cohesive per say but to accomplish the basics with the cricket loom, and experiment with different techniques and materials.

Process Note 1 : The set up itself was honestly the most difficult part of this whole project. The precision and care that it takes just to thread the heddles and wind onto the warp beam was extensive.

Process Note 2: I wanted my first swatch to be a way for me to get used to the loom, so I experimented with how the swatch looked based on how hard I would beat the wefts. You can see the variations.

Process Note 3: For my next swatch, I was inspired by the area I was sitting in outside with my cat. Using yarn I found that is used to make sponges and dish rags, the texture and strength of the yarn pulled inmy warp a bit and I think ultimately made the swatches different sizes. To make the clouds, I came in from the sides and wove in the yarn I was using to make the headers. The contrast of the pokey blue threads with the soft white was very pleasing to me.

Process Note 4: In my final swatch, I wanted to switch the methods with which I was weaving with. So I hand wove every other on some on the bottom shed, then switched for the reverse shed.

Here are the finished swatches ! Stitched off on either end to keep the whole thing from unraveling

Reflection : I really enjoyed using the cricket loom this time. It was so much easier to keep things in place and for the most part keep even tension. I do need to make sure next time not to pull in the warp as much.
